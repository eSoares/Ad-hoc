#Readme

This project tries to enable Ad-Hoc network mode on Android devices.

---
##Working on:
- Optimus Boston (CyanogenMod 7)
- Samsung Nexus S (Cyanogen 10.1)

##Tested and not working on:
- Google Nexus 4
- Google Nexus 5
- Motorola Moto G (2013)
